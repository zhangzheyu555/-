# 卫生复合识别接入说明

状态：现行技术说明（2026-07-15 更新）

督导巡店的「现场拍照识别」已从浏览器直连 Anthropic（原实现缺少 API key，实际不可用）改为调用本地训练的 YOLO 卫生检测模型（Codex 训练产物，检出地面纸屑/污点/角落积灰并画红圈）。

## 调用链

```
Vue3 正式前端（督导巡店 → 发起巡检）
开发地址：:5173/operations/inspection/tasks
        │  POST /api/inspections/detect (multipart file)
        ▼
Spring Boot 后端 InspectionController
        │  转发 POST {app.inspection.detect-url}
        ▼
Python FastAPI 识别服务 (127.0.0.1:8000/detect)
        │  YOLO models/floor_litter_seed_best.pt
        ▼
返回：review_status/deduction_content/deduction_score/annotated_image(红圈图 base64)
```

## 启动顺序

1. 识别服务（已随仓库分发，含模型与 Excel 模板，见 `inspection-service/README.md`）：

```bash
./inspection-service/start.sh      # Mac/Linux
inspection-service\start.bat       # Windows 双击即可
```

（此 Mac 上的原始训练目录在 `~/Documents/Codex/2026-07-06/wo/outputs/kitchen_yolo_review_app`，仓库内是其可部署副本。）

2. 后端（本机无 MySQL 时用 H2 内存库，`application-local.yml` 已被 gitignore，需在本机自建）：

```yaml
# backend/src/main/resources/application-local.yml
spring:
  datasource:
    url: jdbc:h2:mem:store_profit;MODE=MySQL;DATABASE_TO_LOWER=TRUE;CASE_INSENSITIVE_IDENTIFIERS=TRUE;DB_CLOSE_DELAY=-1;NON_KEYWORDS=MONTH,YEAR,DAY,VALUE
    username: sa
    password: ""
    driver-class-name: org.h2.Driver
```

```bash
cd backend
mvn spring-boot:run -Dspring-boot.run.profiles=local   # 有 MySQL 则去掉 profile
```

3. 正式前端：

```bash
cd frontend-vue
npm run dev
```

登录后打开 `http://127.0.0.1:5173/operations/inspection/tasks`。生产环境由 Nginx 托管 `frontend-vue/dist`，不使用 `web`、旧 HTML 或 `runtime-static` 作为业务入口。

## 配置项

`application.yml`：

- `app.inspection.detect-url`（环境变量 `INSPECTION_DETECT_URL`，默认 `http://127.0.0.1:8000/detect`）
- `app.inspection.export-url`（环境变量 `INSPECTION_EXPORT_URL`，默认 `http://127.0.0.1:8000/export`）
- `app.inspection.timeout`（默认 60s）
- 上传限制 20MB（`spring.servlet.multipart`）

## 鉴权说明

浏览器只通过 Vue3 会话携带 Bearer token 调用后端。未登录返回 401，跨角色、跨门店及越权附件访问返回 403。识别服务只接受 Spring Boot 的受控转发，不能被浏览器直接作为业务接口调用。

## Excel 导出

识别完成后，Vue3 页面由后端导出巡检 Excel；浏览器不会从页面 DOM 拼接单据。照片可多张上传，原图、标注结果和条款关联均由受鉴权的后端附件接口保存。

## 注意

- H2 内存库仅供本地开发，重启后数据清空；生产仍是 MySQL。
- 识别服务未启动时接口返回 502「卫生识别服务不可用」。
- 不得将原图、标注图、巡检结果或条款关联写入浏览器 localStorage/KV；真实业务记录以 MySQL 和后端附件存储为准。
