## P0 结论

1. `/profit` 已经是正式业务页，本轮保留并纳入验收。
2. `/store-detail`、`/users`、`/export` 必须移出 `PlaceholderPage`。
3. 手机端不再按小程序/App 拆项目，统一作为 Vue3 响应式 H5 流程继续设计。
4. 不足接口必须明确写“后端待接入”，不能伪造成已保存或已导出。
