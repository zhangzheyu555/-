# Vue3 全角色旧版页面映射 v1

原则：旧版 HTML 是产品原型、UI 标准和交互标准；Vue3 只是技术实现变化。用户看到的菜单名称、页面标题和业务路径应尽量保持旧版一致。真实业务数据继续走后端 API + MySQL，不回退到 localStorage。

读取来源：

- `runtime-static/index.html`
- `backend/src/main/resources/static/index.html`

## 当前结论

1. 菜单名称以旧版为准，Vue3 不再强行套“工作台”名称。
2. 财务、仓库、督导、运营的主要子功能增加独立路由，同时保留旧 query 跳转兼容。
3. 缺后端接口的页面只能显示“后端待接入”，不能假保存。
4. 后续迁移重点是继续把旧 HTML 的页面结构和表单细节拆成 Vue3 组件，而不是重新设计产品。
