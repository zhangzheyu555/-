# 功能模块清单

更新日期：2026-07-18

统计口径：仅统计当前 `frontend-vue` 正式入口和 `backend` 正式业务代码中可独立使用的业务能力。旧路由重定向、兼容页面、构建脚本和测试工具不重复计数。

## 汇总

当前项目共有 **10 个业务域、45 项独立业务能力**。

| 业务域 | 能力数 | 主要使用角色 |
| --- | ---: | --- |
| 角色工作台 | 7 | 老板、财务、仓库、店长、督导、员工 |
| 经营财务 | 6 | 老板、财务、店长 |
| 门店日常 | 3 | 店长、督导、财务 |
| 仓库供应链 | 10 | 仓库、店长、老板、财务 |
| 督导巡店 | 5 | 督导、店长、老板 |
| 员工与培训 | 4 | 员工、店长、老板 |
| 外部平台与配方 | 4 | 老板、督导 |
| 智能助手 | 2 | 授权业务角色、员工 |
| 系统治理 | 3 | 老板 |
| 跨模块闭环 | 1 | 全部授权角色 |

## 1. 角色工作台（7）

1. 老板工作台：经营风险、待办、复核、岗位进度、已完成事项。
2. 财务工作台：财务待办与经营数据处理。
3. 仓库工作台：仓库概览与待处理业务。
4. 店长工作台：本店经营、报销、库存和整改入口。
5. 督导巡店工作台：巡店、问题、复核与标准。
6. 督导工作台：巡检整改、平台配置、培训、盘存等授权范围内的运营辅助功能。
7. 员工工作台：个人门店、工作事项和员工服务入口。

## 2. 经营财务（6）

1. 利润概览。
2. 利润表。
3. 经营数据录入与导入。
4. 每日报销：新增、编辑、提交、图片附件、补充材料、审核、驳回和上报老板。
5. 员工工资与本店工资核对。
6. 经营数据导出。

## 3. 门店日常（3）

1. 每日报损：品类分类选择、数量、原因、图片、提交、督导复核和月度图片导出。
2. 门店详情与本店经营结果查看。
3. 门店档案与门店管理。

## 4. 仓库供应链（10）

1. 物料档案与商品启停。
2. 总仓和区域仓库存。
3. 门店库存。
4. 门店叫货。
5. 仓库处理叫货与发货。
6. 门店收货确认。
7. 外部采购。
8. 采购入库与入库记录。
9. 出入库记录、仓间调拨和库存预警。
10. 配送退货单、附件与打印下载。

## 5. 督导巡店（5）

1. 发起巡检任务。
2. 巡检记录与问题登记。
3. 稽核标准维护。
4. 门店整改。
5. 督导整改复核。

## 6. 员工与培训（4）

1. 员工档案。
2. 员工个人资料、所属门店和工资信息。
3. 培训学习与考试。
4. 培训进度与考试管理。

## 7. 外部平台与配方（4）

1. 企迈开放平台凭证与品牌配置。
2. 企迈营业额查询与导出。
3. 企迈商品销量查询与导出。
4. 配方销量填充、用量测算和水果采购毛重测算。

## 8. 智能助手（2）

1. 门店经营助手。
2. 员工服务助手：问答、反馈和人工转接闭环。

## 9. 系统治理（3）

1. 账号、角色、权限码和数据范围管理。
2. 操作日志与审计查询。
3. 组织、品牌、门店与平台级配置。

## 10. 跨模块闭环（1）

1. 今日待办与业务跳转：将报销、报损、巡检、仓库等异常统一推送到角色待办，并回到对应业务页面处理。

## 技术支撑能力（不计入 45 项）

- 身份认证、会话和路由守卫。
- 多租户、角色权限、门店数据范围和跨门店拦截。
- 附件上传、鉴权预览、下载与图片导出。
- `operation_log` 审计：关键操作、导出、上传下载和权限拒绝。
- Flyway MySQL/H2 数据库迁移、后端事务和并发保护。
- 移动端适配、前端构建与角色菜单验收脚本。

## 不纳入正式功能统计的内容

- `/legacy` 和旧 HTML 兼容入口。
- 路由重定向，例如旧巡检、旧工资和旧培训链接。
- `runtime-static`、`web`、顶层旧版 `index.html`、`database.js`。
- `output` 内截图、临时数据库包、构建产物和验收输出。
