# Vue3 页面级旧版复刻清单 v1

## 目标

旧版 HTML 是产品原型、UI 标准和交互标准。Vue3 当前阶段不重新设计产品，只把旧版页面按现有后端 API / MySQL 数据重新实现。

## 后续阶段

1. 财务和导出页复刻：`/expenses`、`/salary`、`/export`。
2. 仓库、督导、运营内部页面复刻：商品档案、采购入库、巡检记录、考试系统、店铺盘存等。
3. 手机端 H5 适配：375、390、430、768 宽度下保持旧版业务路径可用。

## 数据规则

- 不使用 localStorage 保存真实业务数据。
- 页面只读、预览、测算可以在前端完成；一旦标记为“保存”“提交”“审核”，必须走后端接口并写 MySQL。
- 缺后端接口时，按钮必须显示“待接入”或禁用，不允许假保存。
